# Hướng dẫn sử dụng Obsidian Plugin hiệu quả

Obsidian là một công cụ ghi chú mạnh mẽ, và sức mạnh thực sự của nó đến từ hệ sinh thái **community plugins** phong phú. Bài viết này tổng hợp các plugin hữu ích, phân loại theo nhu cầu sử dụng để bạn dễ dàng lựa chọn.

## Cách cài đặt Community Plugin

1. Mở **Settings** (bánh răng góc dưới bên trái)
2. Vào **Community plugins** → **Turn on community plugins**
3. Nhấn **Browse** để tìm kiếm plugin
4. Chọn plugin → **Install** → **Enable**

> Hầu hết community plugins đều mã nguồn mở trên GitHub, bạn có thể kiểm tra source code trước khi dùng.

## Soạn thảo văn bản

| Plugin | Mô tả |
|--------|-------|
| **Outliner** | Quản lý danh sách (bullet/ordered) trực quan, thu gọn/mở rộng theo mục |
| **Smart Typography** | Tự động định dạng ghi chú — smart quotes, em dash, v.v. |
| **Advanced Tables** | Thao tác bảng Markdown nhanh — thêm/xoá cột, sắp xếp |
| **Data Loom** | Bảng tính phong cách Notion, kéo thả, nhóm dữ liệu |
| **Mind Map** | Tạo mind map từ bullet points (cần mở preview) |
| **Mermaid Tool** | Vẽ flowchart/sequence/gantt bằng cú pháp Mermaid ngay trong Obsidian |
| **Banners** | Thêm banner và avatar cho mỗi ghi chú |
| **Natural Language Dates** | Gõ `@today`, `@time: now` để chèn ngày giờ bằng ngôn ngữ tự nhiên |
| **Emoji Shortcodes** | Gõ `:wink` để chèn 😉 — tương tự Slack/GitHub |
| **Note Refactor** | Tách một ghi chú lớn thành nhiều ghi chú nhỏ tự động |
| **Templater** | Tạo template mạnh mẽ với JavaScript — có thể tự động fill metadata, liên kết, v.v. |
| **QuickAdd** | Capture nhanh ý tưởng vào inbox mà không cần rời tay khỏi bàn phím |

## Ghi chú cá nhân (Journal / PKM)

| Plugin | Mô tả |
|--------|-------|
| **Calendar** | Hiển thị lịch ở thanh bên, click để tạo/mở ghi chú theo ngày |
| **Periodic Notes** | Quản lý daily/weekly/monthly/quarterly notes — tích hợp với Calendar |
| **Daily Notes** | Built-in — tạo ghi chú hàng ngày nhanh chóng |
| **Templater** | Dùng template riêng cho daily notes, tự động chèn metadata, thời tiết, v.v. |

## Ghi chú trực quan

| Plugin | Mô tả |
|--------|-------|
| **Excalidraw** | Vẽ sketch, doodling, diagram ngay trong Obsidian — hỗ trợ desktop & mobile |
| **Obsidian Canvas** | Built-in — tạo canvas trực quan, kết nối các ghi chú bằng đồ thị |

## Quản lý thời gian & dự án

| Plugin | Mô tả |
|--------|-------|
| **Tasks** | Quản lý to-do list từ tất cả ghi chú với query syntax mạnh mẽ |
| **Reminder** | Đặt lời nhắc cho task (hỗ trợ natural language date) |
| **Flexible Pomodoro** | Đồng hồ Pomodoro trong Obsidian + log lịch sử |
| **Kanban** | Tạo kanban board quản lý dự án kiểu Trello |
| **Obsidian TODO | Text-based GTD** | Danh sách việc cần làm gom từ mọi ghi chú |

## Truy vấn & Tổ chức

| Plugin | Mô tả |
|--------|-------|
| **Dataview** | Query ghi chú như database — lọc, sắp xếp, tổng hợp metadata |
| **Omnisearch** | Tìm kiếm toàn văn nhanh hơn search mặc định, hỗ trợ fuzzy |
| **Tag Wrangler** | Quản lý tag — rename, merge, xoá hàng loạt |
| **Folder Notes** | Click vào folder để mở một ghi chú tổng quan thay vì expand |
| **Homepage** | Thiết lập trang chủ tuỳ chỉnh khi mở Obsidian |

## Tinh chỉnh giao diện

| Plugin | Mô tả |
|--------|-------|
| **Hider** | Ẩn các thành phần UI bất kỳ, gán phím tắt bật/tắt |
| **Icon Folder** | Thêm icon cho folder/file — nhìn nhanh hơn đọc tên |
| **Style Settings** | Tuỳ chỉnh CSS toàn bộ giao diện (màu sắc, font, spacing) |
| **Minimal Theme Settings** | Tinh chỉnh theme Minimal — tuỳ biến style heading, accent color |
| **Sliding Panes (Andy Mode)** | Hiển thị nhiều ghi chú dạng tab ngang, như Andy Matuschak |
| **Commander** | Thêm button, menu, ribbon theo ý muốn |

## Xuất bản & Đồng bộ

| Plugin | Mô tả |
|--------|-------|
| **Pandoc Plugin** | Xuất Markdown ra PDF, DOCX, LaTeX, EPUB — yêu cầu cài `pandoc` |
| **Obsidian Git** | Tự động backup vault lên GitHub, hỗ trợ restore |
| **Readwise Official** | Đồng bộ highlight từ Kindle, web, Pocket vào Obsidian |

## Mẹo kết hợp plugin

- **Dataview + Periodic Notes**: Tạo mục "On this day" tự động hiển thị ghi chú từ các năm trước
- **Templater + QuickAdd**: Luồng capture nhanh — một phím tắt tạo ghi chú mới với template định sẵn
- **Kanban + Tasks**: Dùng Kanban cho dự án lớn, Tasks cho việc hàng ngày
- **Calendar + Periodic Notes + Daily Notes**: Hệ thống journal hoàn chỉnh

## Kết luận

Obsidian plugins biến một ứng dụng ghi chú đơn thuần thành một hệ thống quản lý tri thức cá nhân (PKM) thực thụ. Hãy bắt đầu với vài plugin phù hợp nhu cầu, đừng cài quá nhiều cùng lúc — bạn luôn có thể mở rộng dần.
