# Danh sách một số Chat AI miễn phí mà Cẩm nang NQDEV tìm được

Thị trường AI hiện nay có rất nhiều nhà cung cấp cho phép truy cập miễn phí các mô hình ngôn ngữ lớn (LLM) qua API hoặc giao diện chat trực tiếp. Dưới đây là danh sách các dịch vụ Chat AI miễn phí đáng dùng nhất mà Cẩm nang NQDEV đã tìm được.

## 1. GLM Coding — Z.ai

[Z.ai](https://z.ai) cung cấp các mô hình GLM với mức giá **hoàn toàn miễn phí** cho cả input, output và cached read.

| Model | Model ID | Input | Output | Cached Read |
|---|---|---|---|---|
| **GLM-4.7-Flash** | `glm-4.7-flash` | Free | Free | Free |
| **GLM-4.5-Flash** | `glm-4.5-flash` | Free | Free | Free |
| **GLM-4.6V-Flash (Vision)** | `glm-4.6v-flash` | Free | Free | Free |

Chi tiết: [docs.z.ai/guides/overview/pricing](https://docs.z.ai/guides/overview/pricing)

## 2. Free LLM API Resources

Trang [freellm.net](https://freellm.net) tổng hợp các nhà cung cấp LLM miễn phí và các model free:

- [Danh sách Providers](https://freellm.net/providers/)
- [Danh sách Models miễn phí](https://freellm.net/models/?free=1)
- [Free LLM API Keys 2026 — 234+ Free AI Models](https://freellm.net/free-llm-api-keys)

## 3. LLM7.io

| **Docs** | [freellm.net/providers/llm7-io](https://freellm.net/providers/llm7-io) |

## 9. Qwen Chat

[chat.qwen.ai](https://chat.qwen.ai) — giao diện chat miễn phí từ Alibaba Cloud với các mô hình **Qwen** (Qwen2.5, QwQ). Hỗ trợ đa ngôn ngữ, đọc file, xử lý code, và tìm kiếm web. API tại [console.aliyun.com](https://console.aliyun.com).

## 10. DeepSeek Chat

[chat.deepseek.com](https://chat.deepseek.com/) — chat miễn phí với **DeepSeek-V3** và **DeepSeek-R1** (reasoning), thuần Trung Quốc nhưng hỗ trợ tiếng Anh và code rất tốt. Không yêu cầu API key cho chat cơ bản. API tại [platform.deepseek.com](https://platform.deepseek.com/).

## 11. LLM7 Chat

[llm7.chat](https://llm7.chat/) — giao diện chat tích hợp sẵn từ LLM7.io, truy cập nhiều model AI qua một cửa sổ chat duy nhất. API key tại [dash.llm7.io](https://dash.llm7.io/) và base URL `https://api.llm7.io/v1`.

## 12. Mistral AI

[Mistral AI](https://chat.mistral.ai) cung cấp giao diện chat trực tuyến miễn phí với các mô hình Mistral mạnh mẽ như **Mistral Large**, **Mistral Small**, **Codestral** (mã hóa), và **Le Chat** — hỗ trợ cả văn bản lẫn hình ảnh.

| Thông tin | Chi tiết |
|---|---|
| **Chat Interface** | [chat.mistral.ai](https://chat.mistral.ai) |
| **API** | [console.mistral.ai](https://console.mistral.ai) |
| **Docs** | [docs.mistral.ai](https://docs.mistral.ai/) |

## Tổng kết

Hầu hết các dịch vụ trên đều hỗ trợ **OpenAI-compatible API**, nghĩa là bạn có thể dùng chung code/client với OpenAI — chỉ cần đổi base URL và API key. Đây là lợi thế lớn khi muốn thử nghiệm nhiều model khác nhau mà không cần sửa đổi code.

**Lưu ý:** Các dịch vụ miễn phí thường có giới hạn về số request/phút (rate limit) và số token tối đa. Kiểm tra kỹ trước khi dùng cho production.
